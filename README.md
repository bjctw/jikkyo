# Jikkyo #

ニコニコ動画から取得したコメントファイルや、Twitterのタイムラインを、透明なウィンドウの上にニコニコ動画コメント風に流せる実況閲覧アプリです。

Windows / Mac / Linux で動作します。

## For Developers

```sh
git clone https://github.com/rot1024/jikkyo.git
cd jikkyo
npm i
cd src
npm i
npm start

# Building
gulp release
```
